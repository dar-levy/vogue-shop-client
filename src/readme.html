Vogue Clothes - Readme

Project Overview

This project was developed individually and collaboratively with Ziv Somech. The project involves creating an online store named "Vogue Clothes" where customers can browse and purchase fashionable clothing items.

Store Name

The store is called Vogue Clothes.

What Are We Selling?

We are selling fashionable clothing items under the brand name "Vogue Shop".

Additional Pages Added

We added several additional pages to enhance the user experience:

  • About
  • Contact
  • New Arrivals
  • Reviews

These pages can be accessed by clicking on the corresponding links in the navbar header. They operate just like any other page, displaying their contents when selected.

Challenges Faced

One of the hardest parts of this project was storing data globally using React Store Context. Ensuring data was available across different components and routes was a challenging but rewarding task.

Supported Routes

The application supports the following routes:

  • /catalog - Displays the catalog of available products.
  • /catalog/:id - Displays detailed information about a specific product.
  • /about - Displays information about the store.
  • /contact - Provides contact details and a form for inquiries.
  • /reviews - Displays customer reviews.
  • /new-arrivals - Shows the latest products added to the store.
  • /new-product (Admin Only) - Allows adding a new product to the catalog.
  • /activity-history (Admin Only) - Displays the activity history.
  • /basket - Shows the current items in the shopping basket.
  • /checkout - Handles the checkout process.
  • /thank-you/:id - Displays a thank you message after a successful purchase.
  • /logout - Logs the user out of the application.
  • /not-found - Displays a 404 error page when a route is not found.
  • /register - Allows new users to create an account.
  • /login - Allows existing users to log in.
  • / - Redirects to the /catalog route if logged in, or /login if not.
